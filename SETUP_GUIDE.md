# 🤖 AI Task Tracker Bot - Полная инструкция по установке

## 📋 Что это за бот?

Умный Telegram-бот для управления задачами с искусственным интеллектом.

**Возможности:**
- ✅ Добавление задач голосом или текстом
- 🤖 AI автоматически определяет приоритет, категорию и дедлайн
- 📅 Просмотр задач на конкретные дни
- 🌅 Утренний дайджест в 9:00
- ⏰ Напоминания за час до дедлайна
- 📊 Статистика и фильтры

**Ваши категории:**
- Встречи, Личное, Работа, IPG, Китаец, Сиклисити, Синицы, Блог, Покупки, Отдых

**Приоритеты (матрица Эйзенхауэра):**
- 🔴 Важное и срочное
- 🟠 Важное, не срочное
- 🟡 Срочное, не важное
- 🟢 Не важное, не срочное

## 🚀 Пошаговая инструкция для новичков

### Шаг 1: Получение API ключей

#### 1.1 Создание Telegram бота

1. Откройте Telegram и найдите бота **@BotFather**
2. Отправьте команду `/newbot`
3. Придумайте имя бота (например: "My Task Manager")
4. Придумайте username (например: "my_tasks_bot")
   - Username должен заканчиваться на `bot`
   - Пример: `mytasks_bot` или `my_task_tracker_bot`
5. **BotFather пришлёт вам TOKEN** - это длинная строка вида:
   ```
   1234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567890
   ```
6. **СОХРАНИТЕ ЭТУ СТРОКУ!** Это ваш `TELEGRAM_BOT_TOKEN`

#### 1.2 Получение API ключа Anthropic (Claude)

1. Перейдите на сайт: https://console.anthropic.com/
2. Зарегистрируйтесь или войдите
3. Перейдите в раздел **API Keys**
4. Нажмите **Create Key**
5. Назовите ключ (например: "Task Bot")
6. **Скопируйте ключ** - это строка вида:
   ```
   sk-ant-api03-xxx...
   ```
7. **СОХРАНИТЕ ЭТУ СТРОКУ!** Это ваш `ANTHROPIC_API_KEY`

**⚠️ ВАЖНО:** 
- Ключи показываются только один раз!
- Никому не показывайте эти ключи
- Для Claude API нужна привязка карты (есть бесплатный кредит $5)

### Шаг 2: Выбор хостинга (куда поставить бота)

Я рекомендую **Render.com** - это бесплатно и просто!

#### Вариант А: Render.com (РЕКОМЕНДУЕТСЯ для новичков)

**Плюсы:**
- ✅ Полностью бесплатно
- ✅ Очень простая настройка
- ✅ Не нужна кредитная карта
- ✅ Работает 24/7

**Минусы:**
- ⚠️ Засыпает после 15 минут неактивности (но просыпается при первом сообщении)

**Инструкция:**

1. Зарегистрируйтесь на https://render.com/
2. Нажмите **New +** → **Web Service**
3. Подключите ваш GitHub (или создайте новый репозиторий)
4. Выберите репозиторий с ботом
5. Заполните настройки:
   - **Name:** `task-bot` (или любое имя)
   - **Environment:** `Python 3`
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `python bot.py`
   - **Instance Type:** `Free`

6. **Добавьте переменные окружения** (Environment Variables):
   ```
   TELEGRAM_BOT_TOKEN = ваш_токен_от_botfather
   ANTHROPIC_API_KEY = ваш_ключ_от_anthropic
   ```

7. Нажмите **Create Web Service**
8. Подождите 5-10 минут, пока всё установится
9. **Готово!** Бот запущен и работает

#### Вариант В: Запуск на своём компьютере (для теста)

**Только для тестирования!** Компьютер должен быть постоянно включен.

**Инструкция:**

1. Установите Python 3.10+ (скачать с https://python.org/)
2. Установите FFmpeg:
   - **Windows:** Скачать с https://ffmpeg.org/download.html
   - **Mac:** `brew install ffmpeg`
   - **Linux:** `sudo apt install ffmpeg`

3. Скачайте файлы бота:
   - `bot.py`
   - `requirements.txt`

4. Откройте терминал/командную строку в папке с ботом

5. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

6. Создайте файл `.env` и добавьте:
   ```
   TELEGRAM_BOT_TOKEN=ваш_токен_от_botfather
   ANTHROPIC_API_KEY=ваш_ключ_от_anthropic
   ```

7. Запустите бота:
   ```bash
   # Windows
   set TELEGRAM_BOT_TOKEN=ваш_токен
   set ANTHROPIC_API_KEY=ваш_ключ
   python bot.py

   # Mac/Linux
   export TELEGRAM_BOT_TOKEN=ваш_токен
   export ANTHROPIC_API_KEY=ваш_ключ
   python bot.py
   ```

8. Если видите `Bot started!` - всё работает!

## 🔧 Настройка и кастомизация

### Изменение времени утреннего дайджеста

В файле `bot.py` найдите строку:
```python
MORNING_DIGEST_HOUR = 9  # 9:00 утра
```

Измените на нужное время (например, `7` для 7:00)

### Изменение часового пояса

В файле `bot.py` найдите строку:
```python
TIMEZONE = 'Europe/Amsterdam'  # Роттердам
```

Полный список часовых поясов: https://en.wikipedia.org/wiki/List_of_tz_database_time_zones

### Добавление категорий

В файле `bot.py` найдите:
```python
CATEGORIES = [
    'Встречи', 'Личное', 'Работа', 'IPG', 'Китаец', 
    'Сиклисити', 'Синицы', 'Блог', 'Покупки', 'Отдых'
]
```

Добавьте или измените категории по вашему желанию.

### Изменение частоты проверки напоминаний

В файле `bot.py` найдите:
```python
interval=900,  # 15 минут в секундах
```

Измените на:
- `300` - каждые 5 минут
- `60` - каждую минуту
- `1800` - каждые 30 минут

## ❓ Решение проблем

### Бот не отвечает

1. Проверьте, что бот запущен на сервере (Render/Railway)
2. Проверьте логи на сервере
3. Убедитесь, что переменные окружения заданы правильно

### Голосовые не распознаются

1. Говорите чётко и не слишком быстро
2. Проверьте, что установлен FFmpeg (для локального запуска)
3. Попробуйте записать голосовое заново

### AI не правильно определяет категорию

Укажите категорию явно:
```
Написать статью, категория блог
Купить продукты, категория покупки
```

### Неправильно распознаётся дата

Используйте более явные формулировки:
```
❌ "завтра" (может не понять)
✅ "завтра 25 мая в 18:00"
✅ "в пятницу в 15:00"
✅ "25.05 в 18:00"
```
