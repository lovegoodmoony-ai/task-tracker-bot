# 🤖 AI Task Tracker Telegram Bot

Умный Telegram-бот для управления задачами с AI-анализом, голосовым вводом и автоматическими напоминаниями.

## ✨ Возможности

- 🎤 **Голосовой ввод** - просто говорите задачу
- 🤖 **AI-анализ** - автоматическое определение приоритета, категории и дедлайна
- 📅 **Просмотр по дням** - задачи на сегодня, завтра, неделю, конкретную дату
- 🌅 **Утренний дайджест** - список задач каждое утро в 9:00
- ⏰ **Напоминания** - за час до дедлайна
- 📊 **Статистика** - отслеживание прогресса
- 🏷️ **10 категорий** - Встречи, Личное, Работа, IPG, Китаец, Сиклисити, Синицы, Блог, Покупки, Отдых
- 🎯 **Матрица Эйзенхауэра** - автоматическое определение важности и срочности

## 🚀 Быстрый старт

### 1. Получите API ключи

**Telegram Bot:**
1. Напишите @BotFather в Telegram
2. Создайте бота командой `/newbot`
3. Сохраните токен

**Anthropic Claude:**
1. Зарегистрируйтесь на https://console.anthropic.com/
2. Создайте API ключ
3. Сохраните ключ

### 2. Деплой на Render.com (бесплатно)

1. Зарегистрируйтесь на https://render.com/
2. Создайте **New Web Service**
3. Подключите этот GitHub репозиторий
4. Настройте:
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `python bot.py`
5. Добавьте переменные окружения:
   ```
   TELEGRAM_BOT_TOKEN = ваш_токен
   ANTHROPIC_API_KEY = ваш_ключ
   ```
6. Нажмите **Create Web Service**

### 3. Готово! 🎉

Найдите бота в Telegram и отправьте `/start`

## 💬 Примеры использования

### Добавление задач

**Текстом:**
```
Встреча с командой IPG в пятницу в 15:00
Написать пост в блог на следующей неделе
Купить продукты сегодня вечером
```

**Голосом:**
Отправьте голосовое сообщение: "Срочно позвонить клиенту завтра в 10 утра"

### Команды

- `/today` - Задачи на сегодня
- `/tomorrow` - Задачи на завтра
- `/date 25.05` - Задачи на конкретную дату
- `/week` - Задачи на неделю
- `/all` - Все активные задачи
- `/stats` - Статистика
- `/help` - Справка

## 📊 Приоритеты

Бот автоматически определяет приоритет по матрице Эйзенхауэра:

- 🔴 **Важное и срочное** - делать немедленно
- 🟠 **Важное, не срочное** - планировать
- 🟡 **Срочное, не важное** - делегировать
- 🟢 **Не важное, не срочное** - исключить

## 🐛 Решение проблем

### Бот не отвечает
- Проверьте логи на Render.com
- Убедитесь что переменные окружения заданы правильно

### Голосовые не работают
- Говорите чётко
- Проверьте наличие FFmpeg (для локального запуска)

### AI неправильно определяет категорию
Укажите явно: "Купить продукты, категория покупки"
